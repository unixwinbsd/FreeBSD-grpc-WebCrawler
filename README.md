# go-crawler

### Install CLI

```bash
cd crawl && go install
```

### Run

Start the server:

```bash
cd server && go run main.go
```

In another terminal run the CLI with commands:

```bash
crawl -start <some-url> # start crawling a url
crawl -stop <some-url>  # stop crawling a url
crawl -list             # view site tree for all crawled urls
```
